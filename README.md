# Atelier CRM

Premium CRM application for an aesthetic medicine clinic, built with Next.js and a modular React interface.

## Included modules

- Executive dashboard
- Patient CRM and clinical profile
- Multi-provider scheduling
- Omnichannel inbox
- Automation flows
- Social media management
- Financial dashboard
- Analytics and attribution
- Compliance and access control

## Run locally

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

## VPS deploy quick path

Install dependencies and build:

```bash
git clone https://github.com/TheRebrandly/clinicadrjoaolima.git
cd clinicadrjoaolima
npm install
npm run build
npm run start
```

For production, use a process manager such as PM2 and put Nginx in front of the app.
